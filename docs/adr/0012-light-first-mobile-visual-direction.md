# Light-first mobile visual direction

Julius 2.0 will move from the current dark slate interface to a light-first mobile product UI inspired by Finanzguru's calm financial command-center feel without copying its identity. The visual system will use off-white surfaces, ink text, a deep purple primary action color, teal or green financial success states, restrained danger color, and softer category colors. UI implementation must use the `impeccable` design workflow and verification to avoid generic AI-looking patterns.
