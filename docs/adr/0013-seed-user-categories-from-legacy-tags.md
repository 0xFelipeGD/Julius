# Seed user categories from legacy tags

Julius 2.0 will seed every User Account with default English Categories mapped from the legacy tag values: `Food` from `Alimentacao`, `Transport` from `Transporte`, `Health` from `Saude`, `Leisure` from `Lazer`, `Housing` from `Habitacao`, `Taxes` from `Impostos`, and `Other` from `Outros`. Existing Transactions will be backfilled to these user-owned Categories so no historical data loses its category.
